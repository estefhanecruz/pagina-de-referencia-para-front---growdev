Introdução ao HTML

O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação que define a estrutura e o significado do conteúdo de uma página web

Principais Tags de formatação:

Tag Strong: é utilizada para marcar conteúdos de grande importância
Tag i: Representa uma parte do texto com uma diferença semântica
Tag em: Representa ênfase importante de seus conteúdos
Tag mark: Representa um trecho dedestaque em um texto
Tag small: Representa comentários em uma fonte menor, como texto de copyright
Tag del: Representa uma parte do texto que foi excluída de um documento
Tag sup: Define um trecho de texto que deve ser renderizado como um sobrescrito.
Tag sub: Especifica uma linha de texto a ser subescrita
